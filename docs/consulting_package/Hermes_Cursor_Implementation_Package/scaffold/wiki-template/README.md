# Wiki Template

Use this template as the starting scaffold for each org wiki rooted at `~/.hermes/profiles/<org>/`.

Minimum required folders:
- `canon/`
- `data/`
- `governance/`
- `knowledge/inbox/`
- `knowledge/drafts/`
- `knowledge/published/`
- `knowledge/reports/specs/`
- `knowledge/reports/published/`
- `knowledge/templates/`
- `knowledge/workflows/`
- `templates/`
- `vocabulary/`
- `archive/`

Minimum required files:
- `index.md`
- `log.md`
- governance pages
- one workflow page
- one report template page
- one report spec page
- one Knowledge Brain ↔ Data Brain page
