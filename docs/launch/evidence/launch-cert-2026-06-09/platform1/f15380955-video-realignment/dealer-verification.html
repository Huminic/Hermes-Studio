Dealer Widget Verification

Emulates a dealer.com page. Click a store to inject its self-hosted launcher (the teal circle, bottom-right). One script tag, no keys — the same snippet a dealer pastes on their site.

Embed snippet (one tag per store)

<script async src="https://studio.huminic.app/widget/dealer/serra-honda.js"></script>

What to check

Teal circle bottom-right → opens "Choose how to connect" with Web Chat / Instant Call Back / Contact Form / Two-Way Video. Web Chat + Contact Form open the live widget in-panel; Instant Call Back posts a lead; Two-Way Video opens a live face-to-face session. No third-party vendor name should appear anywhere.
